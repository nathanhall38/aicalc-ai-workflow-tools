## Overview

aicalc is a browser-based toolkit of AI developer utilities intended for client-side use. It gathers practical helpers for LLM pricing and token counting together with generators for cron expressions and CSP headers, all delivered in a web-oriented interface.

It is built for developers who want fast checks and simple workflows without moving out of the browser. The set also includes privacy-oriented features such as PII redaction, secret scanning, and dependency vulnerability checks, which makes it handy when inspecting prompts, config files, snippets, and project assets during routine work.

---

## Features

- LLM cost and pricing calculator for estimating model usage
- Token counting for prompt and text analysis
- Cron expression generation for scheduling tasks
- CSP header generation for web security policies
- PII redaction for removing sensitive personal data from text
- Dependency vulnerability checks for package review workflows
- Environment secret scanning for exposed keys and tokens
- Browser-based utilities designed for client-side use

## Usage

Run aicalc directly in the browser and move between tools as your task changes:

1. Open the app in your browser.
2. Pick a utility such as pricing, token counting, cron generation, or CSP creation.
3. Paste or type in your input.
4. Inspect the generated result or analysis.
5. Copy the output into your workflow, notes, or project files.

Example workflow:
- Estimate model cost before running a prompt batch
- Count tokens before sending content to an LLM
- Generate a cron string for a scheduled job
- Create a CSP header for a web app
- Scan text or files for secrets, PII, or dependency issues
